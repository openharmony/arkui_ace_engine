1 前端解析

ArkTS 编译器解析 .ets 文件 → 生成组件描述 JSON/字节码 → 包含 @Component struct MyButton 定义

2 组件树构建 (FrameNode)

FrameNode 创建：ColumnNode(id:1) → TextNode(id:2, content:"Click Me") → GestureNode(事件绑定)

3 布局计算 (Layout)

LayoutAlgorithm.Measure() - 计算 Column/Text 尺寸
LayoutAlgorithm.Layout() - 计算位置坐标

4 渲染绘制 (Render)

RenderNode.Draw() - 创建 Skia Canvas → 绘制背景/边框/文字 → 生成 DisplayList

5 上屏显示

SurfaceFlinger / HDI 提交到屏幕 → 屏幕显示 "Click Me" 文字

三、状态变化时的更新流程

// 用户点击 → 状态变化 .onClick(() => { this.message = 'Clicked!'; // @State 变化 })
1 @State setter 被调用

TextPattern.SetContent("Clicked!")

2 标记脏节点

MarkDirtyNode(PROPERTY_UPDATE_TEXT)

3 重新布局

OnDirtyLayoutWrapperSwap() → 重新计算 Text 尺寸

4 重新渲染

RenderNode.ReDraw() → 更新 DisplayList

5 上屏

SurfaceFlinger 合成显示
