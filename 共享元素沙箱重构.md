SetSandBox 的问题
功能不单一：
1. 同时管理计数器（递增/递减/重置）
2. 同时调用 rsNode_->SetSandBox() 操作渲染节点
3. 返回值含义模糊：true/false 表示"是否实际操作了沙盒"，不是操作是否成功
参数混淆：
- onlyCountMode：实际影响的是"是否调用 rsNode_->SetSandBox()"
- force：有两个不同的含义 - 设置时不增计数器，移除时强制重置计数器为0
调用场景：
// 场景1: 设置沙盒
renderContext->SetSandBox(parentPos, !doRegisterSharedTransition_);
// 场景2: 移除沙盒并检查返回值
if (renderContext->SetSandBox(std::nullopt, !doRegisterSharedTransition)) {
    renderContext->UnregisterSharedTransition(targetRenderContext);
}
// 场景3: 强制移除
renderContext->SetSandBox(std::nullopt, !doRegisterSharedTransition, true);
HasAnimatingGeometryTransition
