# NdkBindInputEvent 测试用例归档

## 用例表

| 测试功能            | 预置条件       | 输入                                    | 预期输出        | 是否自动 | 测试结果 |
| ------------------- | -------------- |---------------------------------------|-------------| :------- | -------- |
| 点击事件示例验证    | 设备正常运行   | 进入示例页面，点击按钮出现列表，点击点击事件列表项        | 打印点击事件信息 | 否       | Pass     |
| 触摸事件示例验证    | 设备正常运行   | 进入示例页面，点击按钮出现列表，点击触摸事件列表项        | 打印触摸事件信息 | 否       | Pass     |
| 鼠标事件示例验证    | 设备正常运行   | 进入示例页面，点击按钮出现列表，使用鼠标在鼠标事件列表项上移动、点击        | 打印鼠标事件信息 | 否       | Pass     |
| 悬浮事件示例验证    | 设备正常运行   | 进入示例页面，点击按钮出现列表，使用鼠标移入或移出悬浮事件列表项        | 打印悬浮事件信息 | 否       | Pass     |
|   轴事件示例验证    | 设备正常运行   | 进入示例页面，点击按钮出现列表，使用鼠标滚轮在轴事件列表项上滚动        | 打印轴事件信息 | 否       | Pass     |
| 按键事件示例验证    | 设备正常运行   | 进入示例页面，点击按钮出现列表，点击按键事件列表项使之获焦，然后按下键盘按键        | 打印按键事件信息 | 否       | Pass     |
