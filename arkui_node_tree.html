一、核心节点继承关系

AceType 基类
↓
UINode 节点基类
↓

继承链说明：
  • AceType - 基础类型系统，提供引用计数等基础能力
  • UINode - 所有UI节点的基类，管理父子节点关系
  • FrameNode - 组件节点实体，组合了Pattern、Layout、Render能力
class AceType { // 基类 // 引用计数、智能指针等基础能力 }; class UINode : public AceType { // 节点树管理: AddChild, RemoveChild, GetParent, GetChildren }; class FrameNode : public UINode, public LayoutWrapper { // 组件核心节点 RefPtr<Pattern> pattern_; // 业务逻辑 RefPtr<RenderContext> renderContext_; // 渲染上下文 RefPtr<EventHub> eventHub_; // 事件处理 };
